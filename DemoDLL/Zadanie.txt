Необходимо написать DLL (подключаемую библиотеку), которая экспортирует
три функции:
- вычисление квадратного корня
    один входной параметр Integer
    один выходной параметр Integer
- сравнение двух целых чисел
    два входных параметра Integer
    один выходной параметр Integer
      -1  = значение 1 больше
      0   = значения равны
      1   = значение 2 больше
- реверс строки (строка наоборот)
    один входной параметр String
    один выходной параметр String
     Вход:   Абвгде
     Выход:  едгвбА

DLL - может загружаться динамически и статичиски.

В первом варианте программы - сделать статическую загрузку DLL


В главной программе - три кнопки, нажатия на которые вызывают
ту или иную функцию DLL


Требования к DLL

- не должен использоваться модуль SHAREMEM  (который позволяет передавать строки
как тип String между DLL и  вызываемой программой)
 считаем что нашу DLL может кто-то вызывать например из программы написанной на
 Visual C++

- для передачи типа String  - следует использовать тип PChar - о нем написано
во второй статье

- функция реверса строк и возращаемый ей результат - чуть сложнее чем первые две
Если столкнешься с проблемой - поищи в Интернете "Delphi как вернуть строку из Dll"
